# Install Guide — Engage Center Case Fetcher

> For pilot users

---

## Prerequisites

- Microsoft Edge (latest stable) or Google Chrome
- Access to Engage Center portal

---

## Installation Steps

### 1. Enable Developer Mode

1. Open Edge and navigate to `edge://extensions`
2. Toggle **Developer mode** to ON (bottom left)

### 2. Load the Extension

1. Click **Load unpacked**
2. Select the `engage-center-extension` folder
3. The extension icon should appear in your toolbar

### 3. Pin the Extension

1. Click the puzzle piece icon in the toolbar
2. Find "Engage Center Case Fetcher"
3. Click the pin icon to keep it visible

---

## Usage

1. **Open Engage Center** in a tab and sign in
2. **Open any case** in the portal (this captures your session)
3. **Click the extension icon** to open the popup
4. **Enter a Case ID** and click "Fetch Case"

---

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Session shows "Not detected" | Open a case in Engage Center first |
| Fetch fails with "Session expired" | Re-open a case in Engage Center |
| "Network error" | Check your connection, try again |
